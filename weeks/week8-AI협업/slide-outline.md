# 클로드와 챗지피티를 협업시켜보자 — 두 AI가 함께 만든 8주의 마무리

## Meta
- **Topic**: 캠프 피날레. 지난 7주를 **주차당 1장**으로 돌아보는 wrap-up을 **먼저** 보여주고(복습), 그 wrap-up 슬라이드가 실은 **클로드(제작)와 챗지피티=코덱스(검수)의 협업 산출물**임을 밝힌 뒤 그 과정을 retrospect로 회고한다. GitHub를 무대로 두 회사의 AI가 협업.
- **핵심 대비(데크 축)**: week6 = 같은 Claude끼리 팀(**동종**) ↔ 오늘 = 다른 회사 AI 둘(클로드+챗지피티, **이종**). 같은 AI는 같은 맹점 / 다른 AI는 다른 눈 → **교차검증**. 생산=클로드 · 검수=코덱스 · 판단=사람.
- **자기참조(중요)**: 이 강의 슬라이드 자체가 협업의 결과물. 코덱스 검수 = "클로드의 7주 요약이 **원본 각 주차 슬라이드와 맞는가**" 대조 → 이종 AI 교차검증의 이상적 사례.
- **Target Audience**: week1~7을 누적 수강한 비개발자 KOICA 동료 (채팅vs코드 · 환경 · MCP · 스킬 · git · 에이전트팀 · GitHub까지 직접 해봄)
- **Tone/Mood**: "도구 하나에서 → AI 팀으로. 8주를 마무리하며." 따뜻한 회고 + 마지막 도약. 어렵지 않게.
- **Slide Count**: 14슬라이드 (50분 — wrap-up 7장 빠르게 ~10분 · 협업 회고·개념 ~20분 · 라이브 재현 ~10분 · Q&A·마무리 ~10분)
- **Aspect Ratio**: 16:9 (720pt × 405pt)
- **style**: Bento Grid (week4~7 토큰 그대로). navy #1A1A2E / lime #E8FF3B / teal #4ECDC4 / yellow #FFE66D / pink #FF6B9D / off-white #F8F8F2 / gray #A0A0B0
- **금지**: "학생"(→"동료"/주어 생략). "두 AI가 직접 대화한다"는 오해 금지 — 소통은 GitHub(이슈/PR) 경유.
- **호칭**: "동료" 또는 주어 생략
- **주차 사실**: 캠프 **마지막 주**(week8, 2026.6.29주). 지난주 = GitHub(week7). week6(에이전트팀, 같은 Claude 동종 팀)과 대비. week7 마무리에서 예고한 "역할 반전 — AI가 이슈/검수 제기"의 이행.
- **진행 방식**: 라이브 전체 시연 안 함(50분 제약) — 협업은 **강의 전에 실제로 완성**해두고, 강의는 그 이력을 **retrospect로 회고** + 대표 순간 1~2개만 **라이브 재현**(코덱스 검수).

## 사실 근거 (실제 확인)
- **챗지피티 = Codex**(OpenAI의 터미널/앱 AI 에이전트). `codex` 설치(0.125.0)·"Logged in using ChatGPT" 상태. `codex exec`(비대화형)·`codex review`(코드/PR 검수) 서브커맨드 존재. (week3 곁다리에서 이미 한 번 소개한 Codex)
- **git worktree** = 한 저장소(.git 공유)에 여러 작업폴더. 클로드=원본 작업트리, 코덱스=worktree → 동시에 같은 repo를 만져도 파일 안 부딪힘. **week6 '독립 컨텍스트(오염 방지)'의 git 버전.**
- **두 AI는 직접 대화하지 않음** — GitHub 이슈/PR/리뷰 코멘트가 회사 다른 두 AI의 **공통 언어**(week7에서 배운 그것).
- **코덱스 앱은 사람이 직접 조작** — 클로드(코드 탭)는 코덱스 앱을 제어 못 함. 협업의 절반은 사람의 손(= '사람이 두 AI를 지휘'하는 장면).
- 실습 repo = 이 캠프 자료 repo 자체 `github.com/amnotyoung/lunch-time-claude-camp`. week1~7 슬라이드(HTML) 전부 완성 → 코덱스가 대조할 원본이 실재.

## 역할별 실행 주체
- **클로드(코드 탭, 자동)**: wrap-up 슬라이드 제작 · 단계별 커밋 · push · PR 생성 · 코덱스 의견 반영 · 머지.
- **코덱스(앱, 사람이 지시)**: worktree에서 원본↔요약 대조 검수 → 클로드 PR에 리뷰 의견.
- **사람**: 두 AI를 지휘, 최종 머지 판단.

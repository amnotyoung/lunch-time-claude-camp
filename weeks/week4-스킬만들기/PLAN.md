## 스킬 패키지 구조 (예시 ktm)

```
ktm/                       # user: ~/.claude/skills/ktm/
                           # project: <프로젝트>/.claude/skills/ktm/
├── SKILL.md               # name + description(트리거 키워드) + 워크플로
├── scripts/
│   ├── fetch.py           # 1단계: koica-reg 카테고리 조회
│   ├── mine.py            # 2단계: TF-IDF 키워드 분석
│   ├── viz.py             # 3단계: 막대그래프 + 워드클라우드
│   └── report.py          # 4단계: 마크다운 보고서 생성
├── examples/
│   └── hr_sample/         # 인사규정 결과 견본
└── README.md
```

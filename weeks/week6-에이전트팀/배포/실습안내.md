# 6주차 실습 안내 — 에이전트팀을 운용해보자

서로 의논하는 **검토팀**(예산·규정·성과)을 띄워, 한 사업 자료를 교차 검토하고 하나의 보고서로 합치는 실습입니다.

## 이 폴더에 든 것
| 파일 | 용도 |
|---|---|
| `CLAUDE.md` | 팀 공통 작업 규칙 — 팀원(에이전트)이 자동으로 읽는 기준 문서 |
| `000000016547_01.pdf` | 검토 대상 자료 (온두라스 교사 ICT 사업 기획조사 결과보고서, 95p, 공공누리 공개문서) |
| `koica-reg-mcp.zip` | 규정 대조용 koica-reg (이미 설치돼 있으면 안 써도 됨) |
| `week6-에이전트팀.pdf` | 이번 주차 발표 슬라이드 (11장) |

## 사전 준비 (한 번만)

**1) Claude Code 버전 — v2.1.32 이상**
- 확인: `claude --version` / 낮으면 `claude update`

**2) 에이전트팀 기능 켜기 (실험 기능)**
- Claude Code에 그대로 말하면 됩니다:
  > "settings.local.json에 `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS`를 1로 추가해줘"
- 그러면 아래처럼 설정됩니다:
  ```json
  { "env": { "CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1" } }
  ```
- **설정 후 Claude Code를 재시작** — 설정은 새 세션부터 켜집니다.
- (모든 프로젝트에 켜려면 `~/.claude/settings.json`에 같은 env를 둡니다.)

**3) koica-reg — 규정 대조용 (3·4·5주차에 설치한 그대로)**
- 이미 있으면 추가 작업 없음.
- 없으면 `koica-reg-mcp.zip`을 풀어 두고, 규정 검색은 다음처럼 사용:
  ```
  cd <압축 푼 koica-reg-mcp 경로>
  python3 koica_search.py search "기자재 조달 구매"
  ```

## 실습하기

1. **이 폴더에서** Claude Code를 엽니다(이 폴더의 `CLAUDE.md`를 팀이 자동으로 읽도록).
2. 아래 한 마디를 그대로 입력합니다:

   ```
   이 사업 자료(000000016547_01.pdf)를 에이전트팀으로 검토해줘.
   예산·규정·성과로 나누고, 겹치는 이슈는 팀원끼리 의논해서
   (규정은 koica-reg, 기준은 CLAUDE.md를 따르고)
   하나의 검토 보고서로 합쳐줘.
   ```
   - 역할 배분은 **팀장(메인 Claude)이 알아서** 합니다. 직접 짚어줘도 됩니다.

3. **작동 확인**: 팀원(예산·규정·성과)들이 **서로 메시지를 주고받으면** 소통형 팀이 켜진 것.
   - 그냥 결과만 올라오면 → 분담으로 돈 것(설정/재시작 확인). 발화에 "**팀원끼리 의논해서**"를 넣어 다시 시키세요.

## 한 줄 요약
> 켜고(설정+재시작) → "에이전트팀으로, 팀원끼리 의논해서 검토해줘" → 팀장이 나누고, 팀원이 의논하고, 하나로 합친다.

※ 팀원이 인용한 규정 조문·수치는 단정일 수 있으니, 최종본은 koica-reg `get`/`verify`로 원문을 확인하세요. (CLAUDE.md에도 같은 규칙이 적혀 있습니다.)

## 더 나아가기 — 오늘은 시작일 뿐
오늘 구성한 검토팀은 **즉석·일회성 데모**입니다(개념 익히기엔 충분). 같은 작업을 **반복 운용**하려면 더 많은 설정이 필요합니다:
- 에이전트 정의 파일(`.claude/agents/`) — 재사용
- 전용 스킬 — '검토 방법'을 분리 · 오케스트레이터 — 워크플로우 고정
- QA 에이전트 — 인용·수치 자동 검증 · CLAUDE.md 포인터 — 매 세션 자동 트리거

이 무거운 설정은 **`harness` 같은 플러그인**이 도와줍니다 — 도메인 한 문장 → 에이전트 팀 + 스킬을 한 번에 구성·재사용.
- 설치: `/plugin marketplace add revfactory/harness` → `/plugin install harness` (설치 후 재시작)
- 사용: "하네스 구성해줘" 류 발화로 트리거

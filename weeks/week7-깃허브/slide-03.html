GitHub, 4가지만 알면 된다

🌐

Remote Repository

원격 저장소

인터넷에 있는 공용 저장소 — 내 PC의 그 폴더를 그대로 올린 곳. 모두가 본다

📮

Issue

이슈 = 건의함

"여기 이상해요"라고 알리기. 계정만 있으면 누구나 남길 수 있다

✏️

Pull Request

PR = 수정 제안서

"이렇게 고쳤어요, 받아줄래요?" 내가 만든 수정안을 제출

✅

Merge

머지 = 채택

제안을 진짜 저장소에 합치기. 저장소 주인이 보고 결정

오늘의
한 바퀴

이슈로 알리고 → PR로 고쳐 보내고 → 머지로 채택. 이 한 바퀴를 직접 돌아본다

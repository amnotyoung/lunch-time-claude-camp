Global Settings vs Project Settings

클로드에게 "이것만은 꼭 지켜줘"를 적어두는 곳

Global

모든 프로젝트에 적용

~/.claude/CLAUDE.md

언어 · 말투

"항상 한국어로 대답해줘"
"말 끝에 이모지 쓰지 마"

코드 습관

"커밋 메시지는 영어로"
"주석은 꼭 필요할 때만"

내 정보

"나는 비개발자야, 쉽게 설명해줘"

Project

이 프로젝트에만 적용 · 팀 공유 가능

.claude/CLAUDE.md (프로젝트 폴더 안)

기술 스택

"이 프로젝트는 Python 3.11"
"패키지 관리는 pip 사용"

규칙 · 제약

"API 키는 .env에서 읽어와"
"tests/ 폴더에 테스트도 만들어"

팀 컨텍스트

"배포는 `python deploy.py`로"

코드 탭에서 바로 열기 → ~/.claude/CLAUDE.md 파일 열어줘

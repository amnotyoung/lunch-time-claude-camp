채팅 탭 vs 코드 탭

코드 탭은 따로 설치할 게 없습니다

채팅 탭

대화 · 문서 작업 중심

내 컴퓨터의 파일과 터미널을 직접 다루려면 별도 연결 도구를 설치해야 합니다

파일 · 터미널 작업 = 추가 설치 필요
(그 도구 이야기는 3주차에서)

코드 탭

우리가 쓰는 탭

파일 읽기·쓰기, 터미널 명령 실행이 처음부터 내장 — 추가 설치 없이 바로 내 PC에서 작업합니다

✓

파일 열고 읽고 수정

✓

터미널 명령 실행

✓

Node · Python · git 설치도 한 줄

이 캠프는 코드 탭만 사용 — 그래서 따로 설치할 도구가 없습니다
